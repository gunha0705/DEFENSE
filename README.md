게임 DEFENSE 설명   
============   
1. 게임 목적   
오른쪽 끝에서 몰려오는 적들을 막아내는 게임입니다.   
   
2. 조작 방법
왼쪽 Ctrl키 => 총알 발사   
화살표 방향키 => 이동   
   
*초록색 영역 안에서만 움직일 수 있습니다.*   
   
3. 세부 규칙   
적은 3번 공격에 적중하면 사라지고 점수가 1점 오릅니다.   
점수가 오를수록 적들이 다가오는 속도가 빨라집니다.   
적들이 왼쪽 끝에 있는 성에 부딫히면 성의 내구도가 1 닳습니다.   
성의 내구도가 0이 되면 게임오버됩니다.   
점수를 100점을 달성하면 성의 내구도가 1 복구됩니다.
